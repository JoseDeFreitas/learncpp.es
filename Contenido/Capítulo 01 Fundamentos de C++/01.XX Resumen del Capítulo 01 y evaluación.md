## Resumen corto

Una **declaración** es una tipo de instrucción que le dice al programa qué hacer. Se suele
poner un punto y coma al final de las declaraciones.

Una **función** es una colección de declaraciones que se ejecutan de manera secuencial. Cada
programa de C++ debe incluir una función especial llamada *main*. Cuando ejecutas tu programa,
este empieza al principio de la función *main*.

Las reglas que dictan cómo debe escribirse código en C++ se llaman **sintaxis**. Un
**error de sintaxis** es un error producido por el incumplimiento de alguna regla gramatical
del lenguaje.

Los programadores pueden usar **comentarios** para dejar notas a lo largo del código. En C++
hay dos tipos de comentarios: comentarios de una línea (que empiezan con //) y van hasta el
final de una sola línea y los comentarios múltiples (que empiezan con /\* y terminan con \*/),
que van desde el símbolo de apertura (/\*) hasta el símbolo de cierre (\*/)No pongas un comentario dentro de otro.

También, los comentarios son usados para deshabilitar una porción de código temporalmente.
Esta práctica es conocida como "comentar el código" (en vez de "agregar comentarios").

Los **datos** son una secuencia de símbolos que tienen un significado. Se le llama **valor**
a un solo dato que está almacenado en alguna locación de la memoria.

Una **variable** es una pieza de memoria con nombre que se usa para almacenar valores. El
nombre de una variable se llama **identificador**. Para crear una variable debemos usar una
declaración llamada **declaración de definición**. Cuando ejecutamos el programa, cada una
de las variables son **inicializadas**, es decir, son añadidas a una dirección en la memoria.

Un **tipo de dato** le dice al compilador cómo interpretar un valor. Un **entero** es un
número no decimal, como 4, 27, 0, -2 o -12.

Podemos usar la **asignación de copia** (con el operador =) para asignarle un valor a una
variable que ya hemos creado previamente.

Podemos usar la **inicialización** para darle una valor a una variable en el mismo momento
en el que es creada. C++ soporta 3 tipos de inicialización: inicialización de copia,
inicialización directa e inicialización uniforme.

Es preferible usar la inicialización uniforme en relación a los otros tipos de inicialización
y usar la inicialización en relación a la asignación.

A pesar de que puedes definir muchas variables en una sola declaración, es mejor definir
cada variable en una línea diferente (en una declaración diferente).

Usando **std::cout** y el operador << podemos enviar un output a la consola. **std::endl** nos
permite enviar un caracter de nueva línea, forzando a la consola a mover el cursor a la
siguiente línea. Usando **std::cout** y el operador >> podemos tomar un valor como input
(a través del teclado).

Una variable que no tiene valor es llamada **variable no inicializada**. Si intentamos
obtener el valor de una variable no inicializada, obtendremos en consecuencia un
**comportamiento indefinido**, el cual se puede manifestar de diferentes maneras.

C++ contiene unas palabras especiales, llamadas **palabras reservadas**. Estas palabras tienen
un significado especial en el lenguaje (¡no las uses como nombres de variables!).

Una **constante literal** es un valor fijo que se escribe directamente en el código fuente.
Algunos ejemplos de una constante literal pueden ser: 5 y "¡Hola, mundo!".

Una **operación** es un cálculo matemático que consta de cero o más valores como input,
llamados **operandos**. La operación que se lleva a cabo entre los operandos es determinada
por el **operador**. El resultado de este cálculo devuelve un valor como output.

Los operadores **unarios** usan un solo operando. Los operadores **binarios** usan dos
operandos (también llamados operando de la izquierda y operando de la derecha). Los operadores
**ternarios** usan tres operandos.

Una **expresión** es una combinación de literales, variables, operadores y llamadas de
funciones que se evalúan para producir un output. Este proceso es llamado **evaluación**.
El valor devuelto se llama **resultado**.

Una **declaración de expresión** es una expresión convertida en una declaración al colocar un
punto y coma al final de la expresión.

La programación es difícil, por lo que es raro que tus programas sean perfectos (o casi
perfectos) a la primera vez. Haz que tus programas funciones y luego refina las partes que
consideres.

## Evaluación

**Pregunta 1**

¿Cuál es la diferencia entre inicialización y asignación?

<details>
    <summary>Mostrar solución</summary>
    La inicialización le otorga a la variable un valor inicial cuando ésta es creada. En cambio, la asignación le da un valor nuevo a una variable que ya ha sido creada previamente.
</details>

**Pregunta 2**

¿Cuándo ocurre un comportamiento indefinido? ¿Cuáles son las consecuencias de un
comportamiento indefinido?

<details>
    <summary>Mostrar solución</summary>
    Un compartamiento indefinido ocurre cuando el programador hace algo que C++ no soporta. Las consecuencias pueden ser cualquiera, desde una falla en el programa hasta un output erróneo.
</details>

**Pregunta 3**

Escribe un programa que le pide al usuario que escriba dos números. El programa debe decirle
al usuario cuál es el resultado de la suma y la resta de ambos números.

El output del programa debe ser igual al siguiente (asumiendo en este ejemplo que el input
es 6 y 4):
```
Escribe un entero: 6
Escribe otro entero: 4
6 + 4 es 10.
6 - 4 es 2.
```

<details>
    <summary>Mostrar solución</summary>

    #include <iostream>

    int main()
    {
        int x{};
        std::cout << "Escribe un entero: ";
        std::cin >> x;

        int y{};
        std::cout << "Escribe otro entero: ";
        std::cin >> y;

        std::cout << x << " + " << y << " es " << x + y << ".\n";
        std::cout << x << " + " << y << " es " << x - y << ".\n";

        return 0;
    }

</details>
